About me

Welcome to SamSherar.co.uk: the portfolio and online hub of Sam Sherar - a second year Undergrad studying MEng Software Engineering at Aberystwyth University. Computing has been a passion of mine since I was a little nipper playing educational games on Windows 3.1 and my thirst hasn't stopped since!

My main interests are mainly software side: programming problems and bugs have always facinated me! From websites not displaying correctly, to reducing database sizes, it has always been a great way to challenge myself.

Some of the programming languages I currently know are:

  • C
  • PHP/MySQL
  • xHTML/HTML5/CSS
  • Java
  • Perl
  • Haskell
  • C#/VB.NET
